## CLS

### Purpose
Clears the display screen.

### Example
```basic
CLS
```

### Explanation
The screen is cleared and the cursor is located at the home position.
Pressing the `SHIFT` `CLS` key or executing ```PRINT CHR$(12);``` produces
 the same result.

### Sample Program
```basic
10 REM CLEAR SCREEN
20 CLS
```

Clears screen.
